## Available Packages

Magicyan is organized as a monorepo with multiple specialized packages:

### @magicyan/core
General-purpose utility functions for everyday TypeScript development including date formatting, validation, math operations, and async utilities.

### @magicyan/discord
Helper functions and utilities specifically designed for Discord.js bot development, including command handlers, embed builders, and component creators.

### @magicyan/discord-events
Extended event system for Discord.js with additional event types and enhanced event handling capabilities.

### @magicyan/discord-ui
Pre-built UI components for Discord messages, making it easy to create beautiful and interactive bot interfaces.

### @magicyan/canvas
Canvas manipulation utilities including gradient generation, image filters, and custom font management.

### @magicyan/minecraft
Integration with various Minecraft APIs for fetching player skins, server status, and Mojang API data.

## Why Magicyan?

Building Discord bots and web applications often involves writing the same utility functions over and over. Magicyan eliminates this repetition by providing:

- **Battle-tested utilities** that have been refined through real-world usage
- **Consistent APIs** across all packages for a smooth developer experience
- **TypeScript-first design** with excellent type inference and editor support
- **Zero-config setup** - install and start using immediately
- **Modular architecture** - only bundle what you actually use

<Note>
All packages are compatible with Node.js 18+ and work seamlessly with Discord.js v14.
</Note>
