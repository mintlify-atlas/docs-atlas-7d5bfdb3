## Introduction

The Magicyan Discord package provides a powerful set of utilities to simplify Discord.js bot development. It offers type-safe helpers for working with commands, channels, members, roles, emojis, webhooks, embeds, and UI components.

## Installation

<CodeGroup>

```bash npm
npm install @magicyan/discord
```

```bash pnpm
pnpm add @magicyan/discord
```

```bash yarn
yarn add @magicyan/discord
```

</CodeGroup>

## Quick Example

Here's a simple example showing multiple utilities working together:

```typescript
import { 
  findCommand, 
  commandMention,
  findChannel,
  createEmbed,
  createRow,
  ButtonBuilder,
  ButtonStyle
} from '@magicyan/discord';
import { Client, ChannelType } from 'discord.js';

const client = new Client({ intents: [...] });

client.on('ready', async () => {
  // Find a command
  const pingCommand = findCommand(client).byName('ping');
  
  // Find a text channel
  const announcements = findChannel(guild, ChannelType.GuildText)
    .byName('announcements');
  
  if (announcements && pingCommand) {
    // Create an embed
    const embed = createEmbed({
      title: '🎉 Bot is Online!',
      description: `Try the ${commandMention(pingCommand)} command!`,
      color: '#5865F2',
      timestamp: true
    });
    
    // Create a button row
    const row = createRow(
      new ButtonBuilder({
        customId: 'get_started',
        label: 'Get Started',
        style: ButtonStyle.Primary
      })
    );
    
    await announcements.send({ embeds: [embed], components: [row] });
  }
});
```

## Type Safety

All utilities are fully typed and work seamlessly with Discord.js types:

```typescript
import { findChannel } from '@magicyan/discord';
import { Guild, ChannelType } from 'discord.js';

const guild: Guild = ...;

// Type is inferred as GuildTextBasedChannel | undefined
const channel = findChannel(guild, ChannelType.GuildText)
  .byName('general');

// TypeScript knows the exact channel type
if (channel) {
  await channel.send('Hello!'); // ✅ Type-safe
}
```
