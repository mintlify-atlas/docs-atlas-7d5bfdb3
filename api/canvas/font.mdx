# Font API Reference

The `@magicyan/canvas` package provides the `CanvasContextFont` class for managing font properties on canvas contexts with a fluent, chainable API.

## CanvasContextFont Class

### Constructor

```typescript
new CanvasContextFont(
  context: SKRSContext2D,
  props?: Partial<CanvasContextFontProps>
)
```

Creates a new font manager instance that wraps a canvas 2D context.

<ParamField path="context" type="SKRSContext2D" required>
  The canvas 2D context from `@napi-rs/canvas` to manage.
</ParamField>

<ParamField path="props" type="Partial<CanvasContextFontProps>">
  Optional initial font properties to apply immediately.
</ParamField>

**Example:**
```typescript
import { createCanvas } from "@napi-rs/canvas";
import { CanvasContextFont } from "@magicyan/canvas";

const canvas = createCanvas(800, 600);
const ctx = canvas.getContext("2d");

const font = new CanvasContextFont(ctx, {
  family: "Arial",
  size: 24,
  weight: "bold",
  align: "center"
});
```

## Methods

### set()

```typescript
set(props: CanvasContextFontProps): this
```

Update multiple font properties at once. All properties are optional.

<ParamField path="props" type="CanvasContextFontProps" required>
  An object containing the font properties to update:
  
  - `family?: string` - Font family name
  - `size?: number` - Font size in pixels
  - `style?: StrOr<ContextFontStyle>` - Font style (italic, underline, etc.)
  - `weight?: StrOr<ContextFontWeight>` - Font weight (bold, 600, etc.)
  - `align?: CanvasTextAlign` - Text alignment (left, center, right, etc.)
  - `baseline?: CanvasTextBaseline` - Text baseline (top, middle, bottom, etc.)
  - `kerning?: CanvasFontKerning` - Font kerning (auto, normal, none)
</ParamField>

<ResponseField name="returns" type="this">
  Returns the font instance for method chaining.
</ResponseField>

**Example:**
```typescript
font.set({
  family: "Helvetica",
  size: 32,
  weight: "bold",
  align: "center",
  baseline: "middle"
});
```

### setWeight()

```typescript

```

Set the font weight.

<ParamField path="weight" type="StrOr<ContextFontWeight>" required>

</ParamField>

<ResponseField name="returns" type="this">
  Returns the font instance for method chaining.
</ResponseField>

**Example:**

## Type Definitions

### CanvasContextFontProps

```typescript
interface CanvasContextFontProps {
  family?: string;
  style?: StrOr<ContextFontStyle>;
  weight?: StrOr<ContextFontWeight>;
  size?: number;
  align?: CanvasTextAlign;
  baseline?: CanvasTextBaseline;
  kerning?: CanvasFontKerning;
}
```

Interface for font properties used in the constructor and `set()` method.

---

### ContextFontWeight

```typescript
type ContextFontWeight = 
  | "thin" | "extralight" | "light" | "regular" 
  | "medium" | "semibold" | "bold" | "extrabold" | "black"
  | "100" | "200" | "300" | "400" | "500" 
  | "600" | "700" | "800" | "900";
```

Type-safe font weight values, including both named and numeric weights.

---

### ContextFontStyle

```typescript
type ContextFontStyle = 
  | "italic" 
  | "italic underline" 
  | "italic strikethrough"
  | "italic strikethrough underline"
  | "strikethrough" 
  | "strikethrough underline"
  | "underline";
```

Type-safe font style values for text decoration.

---

### StrOr\<T\>

```typescript
type StrOr<Completions> = (string & {}) | Completions;
```

Utility type that provides autocomplete suggestions while still accepting any string value.

## Usage Examples

### Method Chaining

```typescript
const font = new CanvasContextFont(ctx);

font
  .setFamily("Arial")
  .setSize(32)
  .setWeight("bold")
  .setAlign("center")
  .setBaseline("middle");

ctx.fillText("Hello World", 400, 300);
```

### Accessing Properties

```typescript
const font = new CanvasContextFont(ctx, {
  family: "Arial",
  size: 24,
  weight: "bold"
});

console.log(`Font: ${font.family} ${font.size}px`);
console.log(`Weight: ${font.weight}`);
console.log(`Alignment: ${font.align}`);
```

### Reusing Font Configurations

```typescript
const font = new CanvasContextFont(ctx, {
  family: "Arial",
  align: "center",
  baseline: "middle"
});

// Title
font.setSize(48).setWeight("bold");
ctx.fillText("Title", 400, 100);

// Subtitle
font.setSize(24).setWeight("regular");
ctx.fillText("Subtitle", 400, 150);
```
