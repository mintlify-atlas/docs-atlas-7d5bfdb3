## Introduction

The Magicyan Core package is a collection of essential TypeScript utilities designed to streamline Discord.js bot development. It provides a wide range of helper functions for common programming tasks including string manipulation, data conversion, validation, mathematical operations, and date handling.

## Installation

```bash
npm install @magicyan/core
```

```bash
pnpm add @magicyan/core
```

```bash
yarn add @magicyan/core
```

## Quick Start

```typescript
import { 
  capitalize,
  hexToRgb,
  isEmail,
  randomNumber,
  createDate,
  toMs
} from "@magicyan/core";

// String formatting
const title = capitalize("hello world", true);
// "Hello World"

// Color conversion
const color = hexToRgb("#FF5733");
// 16724755

// Validation
const valid = isEmail("user@example.com");
// true

// Random numbers
const dice = randomNumber(1, 6);
// Random number between 1 and 6

// Date manipulation
const futureDate = createDate()
  .add("days", 7)
  .add("hours", 2);

// Time conversion
const fiveMinutes = toMs(5, "minutes");
// 300000
```

## Module Organization

The Core package is organized into logical modules:

- **Conversion** - Color format conversions
- **Formatting** - String builders and text manipulation
- **Validation** - Input validation and type checking
- **Math** - Random number generation and numeric parsing
- **Timers** - Interval and timeout management
- **Date** - Advanced date manipulation with DatePlus
- **Utilities** - Case-insensitive comparisons and function extensions

<Note>
  All functions are fully typed with TypeScript and include comprehensive JSDoc documentation for excellent IDE autocomplete support.
</Note>
